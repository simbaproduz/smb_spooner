Menus

Menu de Spawn - F

O menu de Spawn oferece listas pesquisaveis para selecionar uma entidade para spawnar. Clicar com o botao esquerdo em uma entidade a define como seu spawn atual.

Ao clicar num item em Peds, Veiculos, Objetos ou nos submenus de Pessoal, o menu fecha e um preview local e ativado. Com o preview ativo, use ↑ e ↓ para navegar pela lista atual e E para spawnar o item mostrado.

Se uma entidade nao estiver incluida na lista, ainda e possivel spawnar digitando o nome completo do modelo no campo de busca e clicando em Spawnar por Nome.

Clicar com o botao direito em uma entidade em qualquer menu de spawn adiciona essa entidade aos favoritos. Clicar no botao de favoritos alterna a exibicao para mostrar apenas as entidades favoritadas.

Menu de Banco - X

O menu de Banco armazena uma lista de entidades. Quando uma entidade e spawnada, ela e adicionada automaticamente ao banco atual. Entidades existentes podem ser adicionadas/removidas do banco pelo menu de Propriedades.

  • Clique com o botao esquerdo em uma entidade para abri-la no menu de Propriedades
  • Clique com o botao direito em uma entidade para exclui-la
  • Clique em Excluir Tudo para excluir todas as entidades do banco

Menu de Propriedades - e

O menu de Propriedades lista e permite editar as propriedades de uma entidade.

Propriedades marcadas como (local) nao sao sincronizadas pela rede, entao as alteracoes nelas so serao visiveis para voce.

Menu Salvar/Carregar Banco - J

O menu de Salvar/Carregar Banco permite salvar o banco atual com um nome e carregar todas as entidades novamente depois.

  • Para salvar o banco atual, digite um nome no campo e clique em Salvar.
  • Para carregar um banco salvo, clique com o botao esquerdo no nome do banco.
  • Para excluir um banco salvo, clique com o botao direito no nome do banco.
  • Para importar um banco ou exportar o banco atual, clique em Importar/Exportar.

Marcar a caixa Carregar relativo a posicao do cursor fara com que as entidades do banco selecionado sejam spawnadas relativas a posicao atual do cursor, em vez de exatamente onde foram colocadas originalmente.

Marcar a caixa Substituir banco atual fara com que o banco carregado substitua o banco atual, em vez de mesclar os dois.

Marcar a caixa Salvar/Carregar exclusoes fara com que as entidades excluidas sejam salvas e excluidas novamente quando o banco for carregado.

Menu Importar/Exportar

O menu de Importar/Exportar permite importar e exportar bancos em diferentes formatos:

Formato Descricao Exporta? Importa?
Spooner DB JSON Formato nativo usado pelo spooner Sim Sim
Map Editor XML Formato XML usado pelo editor de mapas Lambdarevolution e pela resource objectloader Sim Nao
Ymap Formato de mapa nativo usado por GTA V/RDR2 Sim Sim
propplacer JSON Banco JSON do propplacer do RedEM:RP Sim Nao
Backup do Spooner Backup de todos os bancos do spooner Sim Sim

Para exportar, selecione o formato desejado e clique em Exportar. A saida sera exibida na caixa de texto, e voce podera copiar para salvar em um arquivo externo.

Para importar, cole o conteudo na caixa de texto, selecione o formato apropriado e clique em Importar. Objetos importados serao adicionados ao banco atual.

Inserir a URL de um arquivo JSON/XML no campo Importar por URL e clicar em Importar permite importar de fontes externas da web, como pastebin.com, sem precisar copiar e colar. Garanta que a URL aponte para a versao raw do arquivo ao usar esse tipo de servico.
